# Roadmap

## Fase 1: Base MVC

- Estrutura `model`, `controller` e `view`.
- API inicial de projetos.
- API inicial de grupos.
- API inicial de oportunidades.
- Tela inicial em Angular.

## Fase 2: Usuarios

- Cadastro.
- Login.
- Perfil academico.
- Edicao de dados.
- Areas de interesse.

## Fase 3: Projetos e grupos

- Catalogo de projetos.
- Pagina de detalhes do projeto.
- Solicitacao de entrada.
- Pagina de grupos.
- Vinculo entre grupos e projetos.

## Fase 4: Gamificacao

- Pontos.
- Badges.
- Rankings.
- Historico de atividades.

## Fase 5: Eventos e oportunidades

- Calendario de eventos.
- Mural de oportunidades.
- Inscricoes ou links externos.
- Filtros por area, tipo e status.

## Fase 6: Painel administrativo

- Moderacao.
- Relatorios.
- Gestao de usuarios.
- Acompanhamento de engajamento.

## Fase 7: Testes
