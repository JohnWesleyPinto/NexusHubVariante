# Refactor Inventory

Inventario da Fase 1 do plano de execucao. Este documento registra o estado atual do projeto antes da refatoracao estrutural de banco, backend e Angular.

Data da analise: 2026-06-07.

## 1. Current Repository State

O projeto esta em estado de transicao:

- `view-angular` foi substituido por `view`.
- `view-next` foi removido.
- O backend possui uma migration inicial de compatibilidade.
- O modelo Java ainda usa entidades de prototipo com `Long` e `GenerationType.IDENTITY`.
- O modelo alvo documentado exige UUID, FKs reais, tabela de humano e tabelas relacionais.

Arquivos modificados antes desta fase devem ser preservados. O agente nao deve reverter alteracoes existentes.

## 5. Current Database Inventory

Migration atual:

- `controller/src/main/resources/db/migration/V1__init_schema.sql`

Tabelas atuais:

- `sec_user`
- `grp_group`
- `opp_opp`
- `prj_project`
- `prj_request`

Problemas estruturais da migration atual:

- Usa `BIGINT GENERATED BY DEFAULT AS IDENTITY`.
- Nao possui `usr_human`.
- Nao possui `sec_role`.
- Nao possui `grp_hummember`.
- Nao possui `prj_hummember`.
- Nao possui `usr_interest`.
- Nao possui `usr_humint`.
- Nao possui `prj_tag`.
- Nao possui `prj_prjtag`.
- Nao possui `opp_apply`.
- Mantem dados soltos como `nmleader`, `nmgroup`, `nmauthor`, `nmuser`, `dsemail`.
- Usa `tscreated`, mas o padrao alvo aprovado e `tsupdated`.
- Nao possui `strecord`, `idupdatedby`, `tsupdated` nas tabelas principais.

## 10. Risks Before Phase 2

- Refatorar Java antes do banco pode gerar entidades desalinhadas.
- Refatorar Angular antes dos DTOs pode gerar retrabalho.
- Remover campos soltos sem ajustar seed e frontend pode quebrar telas atuais.
- Migration atual e entidades atuais estao alinhadas entre si, mas desalinhadas com o modelo alvo.
- O seed atual usa nomes de pessoas/grupos como relacionamento textual.
- A tela de grupo detalhe contem dados locais/mockados que precisam ser separados do dominio real.
- Builds nao foram executados nesta fase porque o inventario nao altera codigo executavel.
